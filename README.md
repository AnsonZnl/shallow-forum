### 前端服务启动 (client文件夹下执行)

- 启动开发服务
```bash
yarn dev 或 npm run dev  
```
- 打包前端页面
```bash
yarn build 或 npm run build  
```
- 打包后执行生产模式服务
```bash
yarn start 或 npm start  
```
